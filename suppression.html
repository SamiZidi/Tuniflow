Suppression des données utilisateur

Tuniflow respecte votre vie privée. Si vous souhaitez supprimer vos données personnelles associées à votre compte Facebook ou à votre interaction avec notre chatbot, vous pouvez en faire la demande à tout moment.

Comment supprimer vos données ?

Pour demander la suppression de vos données, veuillez nous contacter par e-mail à l'adresse suivante :

📧 contact@tuniflow.com

Veuillez inclure dans votre message :

  • Votre identifiant Facebook (ou le nom utilisé dans le chat)
  • La date approximative de l'interaction

Nous traiterons votre demande dans un délai maximum de 7 jours ouvrables.
